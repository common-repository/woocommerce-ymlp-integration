== Description ==

= YMLP integration for your WooCommerce checkout form =

This plugin will add a sign-up checkbox to your WooCommerce checkout form. When customers check this checkbox, they will be subscribed to the YMLP group of your choice.

== Installation ==

= Installing the plugin =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *WooCommerce - YMLP Integration* and click "Install now"
1. Alternatively, download the plugin and upload the contents of `woocommerce-ymlp.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin. 

= Configuring the plugin =
1. Go to *WooCommerce > Settings > Integrations > YMLP Settings* and fill in your YMLP API key and username.
1. Save your settings, a dropdown with your YMLP groups will show up.
1. Select the groups you want to subscribe customers to.
1. *(Optional)* Customize the label text, choose to load default CSS and select whether the checkbox should be prechecked.
